  • accept_date_time (datetime) – filter out ConfigurationSetting created after this datetime

HttpRequestError

async update_configuration_setting(key, value=None, label=None, etag=None, **kwargs)[source]¶

Update specified attributes of the ConfigurationSetting

Parameters

  • key – key used to identify the ConfigurationSetting

  • value – the value to be updated to the ConfigurationSetting. None means unchanged.

  • content_type – the content type to be updated to the ConfigurationSetting. None means unchanged.

  • tags – tags to be updated to the ConfigurationSetting. None means unchanged.

  • label – lable used together with key to identify the ConfigurationSetting.

  • etag – the ETag (http entity tag) of the ConfigurationSetting. Used to check if the configuration setting has changed. Leave None to skip the check.

  • kwargs – if “headers” exists, its value (a dict) will be added to the http request header

Return type

ConfigurationSetting

Raises

ResourceNotFoundError, ResourceModifiedError, HttpRequestError

Example

# in async function
updated_kv = await async_client.update_configuration_setting(
    key="MyKey",
    label="MyLabel",
    value="my updated value",
    content_type=None,  # None means not to update it
    tags={"my updated tag": "my updated tag value"}
)
